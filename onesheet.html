The Intelligent PSD to Native UI compiler.

Turn your Photoshop designs into usable UI code ... not just slicing images, so you can get on to doing what you love.



Have you ever spent a whole day slicing a Photoshop design and coding it to native UI code?

Slicing PSD's is a long, tedious process that cuts into precious development time. Wouldn't you love a product that can take a Photoshop file and generate ready-to-use native UI code for the platform of your choice? Sounds pretty far-fetched, doesn’t it? … Well with AppXen, we’ve done just that. With our simple drag and drop process, you can go from design to UI in seconds. AppXen analyzes your Photoshop file, parses out the necessary data, slices your images into retina & non-retina versions, and generates the UI code for the platform of your choice. This will save a developer an average of 2-8 hours per screen of an application design. With the clean, ready-to-use code, you’ll be on your way to producing apps faster than ever before.


Let Appxen do the work and turn those mock-ups in to running apps.

Select your output platform, drop your PSD file on the app and take a break, but not a long one ... AppXen does this all in seconds. You get back cleanly formatted, ready-to-use native code and all the accompanying images just waiting for you to create something amazing.


Integrate AppXen into your favorite editor.

Using the our custom API you can connect your favorite editor to work with AppXen so you never have to leave what you are doing. Appxen allows you to speed up your workflow and work smarter.


Best of all you don't even need to have Photoshop.

Appxen will read and compiles your PSD all on its own. So you no longer need to own Photoshop just work with all those PSDs.


Features

  • Keep your money in your pocket, Photoshop not required
  • Use from command line or with our easy-to-use GUI
  • Drop a file and get back the clean native code
  • Support for all Photoshop Layer Types
  • Full support for Text Layers and Font Styles
  • Full support for Vector Shapes
  • Outputs optimized PNG images
  • Retina & Non-Retina exports
  • Maintains view hierarchy
  • Maintains layer positioning
  • Extendable functionality

Currently Supported Outputs

  • Appcelerator Titanium: iOS, Android, MobileWeb
  • Alloy for Titanium
  • Native iOS (Objective C)
  • Android (XML)
  • HTML and CSS (That includes PhoneGap)

Templates

Not a fan of the generated code for your selected platform? That's fine, just put in a pull request to make the templates better. All of AppXen's code output templates will be Open-Sourced, allowing for an ever-evolving ecosystem where developers are able to continually create new possibilities.

Current Status

AppXen has been in development as a side project over the past year, being worked on by a team of one. With the proper funding we will be able to dedicate more time to the project and add more brains to our development team. Then we will be hard at work fine-tuning and finishing the remaining pieces: - Creating an extendable Javascript based, plugins system (Yeah, that means Node.js!) - Optimizing the image output to give you crisp images from your PSD (Retina & Non-Retina) - Cutting down compile time on large files to get your code generated faster - Busting out a slicker, feature-rich command line and graphic interface - Bulk compiles, to generate the code for all your screens in parallel
